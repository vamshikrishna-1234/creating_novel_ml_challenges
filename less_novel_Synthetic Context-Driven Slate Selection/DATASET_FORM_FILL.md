## Overview

This dataset is fully synthetic, generated programmatically using NumPy with a fixed random seed (42) for full reproducibility. It simulates a fictional item-selection platform where users are presented with personalized candidate slates and choose one item from each slate based on their preferences and the current context.

The dataset contains 400 users, 800 items, and 20,000 selection queries. Each query records a user being shown a slate of 10 candidate items under specific contextual conditions (6 context features). The user selects exactly one item from the slate. User preference profiles and item characteristics are provided as separate reference tables with anonymized features.

The underlying selection mechanism depends on hidden user preferences that vary with context — users do not have a single fixed preference profile but instead exhibit different preference patterns depending on the combination of contextual conditions. Approximately 12% of selections are noisy (not driven by preferences), introducing an irreducible error floor.

## File Structure

- `users.csv` — 400 user profiles (400 rows, 7 columns). Each row describes one user with anonymized profile features.
- `items.csv` — 800 item descriptions (800 rows, 9 columns). Each row describes one item with anonymized attribute features.
- `queries.csv` — 20,000 selection queries (20,000 rows, 10 columns). Each row records one query: which user, what context, which 10 candidate items (pipe-separated), and which item was chosen.

## Notes

- Each query has exactly 10 candidate items and exactly 1 chosen item.
- The chosen item is always one of the 10 candidates.
- A user may appear in multiple queries with different contexts and slates.
- An item may appear as a candidate in many queries.
- User preferences are context-dependent — the same user may prefer different items under different contextual conditions.
- Approximately 12% of selections are random (not preference-driven).
- All data is synthetically generated with a fixed random seed (42) for reproducibility.
