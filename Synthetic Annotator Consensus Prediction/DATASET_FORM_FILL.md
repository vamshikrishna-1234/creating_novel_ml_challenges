## Overview

This dataset is fully synthetic, generated programmatically by `generate.py` using NumPy with a fixed random seed (42) for full reproducibility. It models a remote monitoring system in which 600 independent telemetry channels report ordinal state readings (0–5) for 12,000 monitored units. A separate expert calibration process establishes a ground-truth reference ordinal state for each unit.

Each telemetry channel has a hidden fidelity profile: its measurement accuracy depends on a latent property of the unit being monitored. This creates a **channel × unit-property interaction** — the same channel may produce accurate readings for units in one latent category but severely degraded readings for units in another category. Channels are organized into 5 hierarchical source groups. Each reading is accompanied by a self-reported confidence score (which is partially informative but adversarially manipulated for corrupted readings) and was collected under one of 4 measurement contexts. Approximately 5% of channels are adversarially miscalibrated. Each unit receives 5–8 independent channel readings in the raw data. The reference state is established through expert calibration with approximately 8% irreducible noise.

The intended prediction task is **robust ordinal state inference**: given a unit's anonymized feature profile, available channel readings (with confidence scores and measurement contexts), and channel profile features (including group membership), predict the expert-calibrated reference ordinal state. The central challenge combines robustness to adversarial confidence manipulation, feature-guided channel × unit interaction modeling, and calibration transfer from high-density training to sparse test conditions.

## File Structure

- `items.csv` — Unit feature profiles for all 12,000 monitored units
- `annotators.csv` — Channel profile features for all 600 telemetry channels
- `annotations.csv` — Individual channel readings (~78,000 rows, 5–8 per unit)
- `consensus.csv` — Expert-calibrated reference ordinal states (12,000 rows)

## Notes

- Each unit has exactly one reference ordinal state.
- Each unit receives 5–8 independent channel readings in the raw data.
- Channel fidelity varies across latent unit categories — the same channel may be reliable for some unit types and unreliable for others.
- All data is synthetically generated with a fixed random seed (42) for reproducibility.
- No real individuals, units, or measurements are represented.
