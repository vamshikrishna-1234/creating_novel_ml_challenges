## Overview

This dataset is fully synthetic, generated programmatically using NumPy with a fixed random seed (42) for full reproducibility. It simulates a fictional reference knowledge base about an invented island nation ("Velantra"), composed of dated documents from multiple named sources, and a set of factual claims that must be verified against this knowledge base as of a specified date.

The knowledge base contains 600 documents spanning the years 2010–2025, authored by 30 distinct sources covering 8 topic categories. Documents describe events, people, locations, and facts about the fictional nation. Some documents correct, retract, or partially modify claims made in earlier documents. Sources have hidden topic-dependent credibility profiles that change over time: a source reliable on one topic may be unreliable on another, and a source's reliability may improve or decay over the years. Approximately 5 sources are adversarial (high surface credibility but periodically inject factual errors), and approximately 15% of sources are "copycat" sources that replicate content from a parent source, creating correlated noise that inflates apparent consensus.

The claims dataset contains 5,200 factual assertions, each with an "as-of" date that determines which documents in the knowledge base are temporally visible for verification. Each claim is labeled with a verification verdict. The verdict depends on the available evidence, source credibility, temporal ordering of documents, presence of corrections or retractions, and whether the evidence comes from adversarial or copycat sources. Approximately 12% of verdicts have irreducible noise reflecting genuine ambiguity.

## File Structure

- `knowledge_base.csv` — 600 dated documents about the fictional nation (600 rows, 9 columns). Each row represents one document with its publication year, source attribution, topic category, associated entities (location, person, event), a reference number, and the full free-text content.
- `claims.csv` — 5,200 factual claims requiring verification (5,200 rows, 4 columns). Each row contains a claim identifier, the claim text, an "as-of" year for temporal filtering, and the verification verdict.

## Notes

- Not all corrections are correct — some introduce new errors (temporal traps).
- All entities, locations, events, and facts are entirely fictional.
- All data is synthetically generated with a fixed random seed (42) for reproducibility.
