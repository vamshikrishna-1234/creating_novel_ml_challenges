Overview
In this challenge, you are given sentences with a missing word represented by ____. Your task is to predict the correct word that completes each sentence.

Each sentence describes a situation involving technical systems and components. The missing word corresponds to a specific component that fits the context of the sentence.

Your goal is to build a model that can accurately determine which word best completes each sentence.

Task
For every sentence in the test dataset, predict the word that should replace the blank (____).

The possible answers belong to a small set of system components such as:

valve
radiator
gasket
bearing
Your submission should contain a predicted component for every test row.

Dataset
The challenge dataset consists of the following files.

train.csv
Contains labeled examples that you can use to train your model.

The correct answers for this file are hidden and will be used to evaluate your submission.

sample_submission.csv
An example submission file showing the required format.

Submission Format
Your submission must be a CSV file with the following structure:

id,answer
abc123,valve
def456,gasket
ghi789,bearing
Requirements:

The file must contain exactly the same IDs as test.csv.
Each ID must appear exactly once.
The answer column must contain your predicted word for that row.
